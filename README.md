# 🛕 Temple Management System

A comprehensive web-based Temple Management System built with Streamlit and Supabase.

## Features

- 🔐 User Authentication with Role-based Access
- 👥 Devotee Management with Bulk Import
- 🧾 Billing and Receipt Generation
- 🙏 Pooja Management and Scheduling
- 💰 Expense Tracking
- 🎁 Donation Management with Receipts
- 🏷️ Asset Management
- 📊 Comprehensive Reports
- 📱 Responsive Design

## Tech Stack

- **Frontend:** Streamlit
- **Database:** Supabase (PostgreSQL)
- **Deployment:** Streamlit Cloud

## Prerequisites

- Python 3.8+
- Supabase account
- Git
